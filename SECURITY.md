# Security Policy

## Reporting a Vulnerability

Please do not report security issues in public issues.

Use GitHub Private Vulnerability Reporting for this repository:

https://github.com/timokoethe/SX1278/security/advisories/new

Include a clear description, affected code or version, reproduction steps, and any relevant hardware or MicroPython details.

## Scope

This project is a MicroPython driver for SX1278/Ra-01 LoRa modules on Raspberry Pi Pico boards. Reports about unsafe defaults, denial of service, unexpected packet handling, or issues that could affect downstream users are welcome.

General bugs and feature requests should use the regular issue templates.
